# Jogo da Bruxa (Witch Game) - A Vision Pro Game

This project is a 2-week prototype built for Apple Vision Pro, combining 3D interactions, spatial design, and gesture-based mechanics.
The experience blends fantasy and technology, allowing users to interact with magic potions in a fully immersive environment created with Reality Composer and 3D assets.
